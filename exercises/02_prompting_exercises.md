# Exercise 2 - Prompting Patterns

## Goal
See how the same text produces different outputs with different prompt styles.

## Tasks
1. Run `python 02_prompt_patterns.py`
2. Add one more prompt pattern of your own:
   - teacher mode
   - interviewer mode
   - recruiter mode
   - strict bullet mode
3. Ask the model to:
   - summarize
   - classify
   - rewrite
   - extract action items

## Challenge
Design a prompt that returns:
- exactly 4 bullets
- each bullet under 10 words
- no introductory sentence

## Discussion
Why is prompt engineering really interface design?
