# Exercise 5 - Chat Memory

## Goal
Understand how conversation history changes output.

## Tasks
1. Run `python 05_chat_memory.py`
2. Ask:
   - "My name is Asha."
   - "I am new to Python."
   - "Recommend a learning path for me."
3. Restart the script and ask only the last question.
4. Compare both answers.

## Discussion
- Did the model remember context only because you sent the history?
- What happens if the history becomes too long?
